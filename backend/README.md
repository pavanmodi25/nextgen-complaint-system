## Setup
1. Copy `.env.example` to `.env` and edit values:
```
MONGO_URI=mongodb://127.0.0.1:27017/college_complaints
JWT_SECRET=your_secret_here
PORT=5000
CLIENT_URL=http://localhost:5173
```
2. Install and run:
```bash
npm install
npm run dev
```
API base: `http://localhost:5000/api`
